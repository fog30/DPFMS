# 🌾 Farm Management System (DPFMS)

## 📌 Overview

The Farm Management System (DPFMS) is a web-based application designed to manage farm operations efficiently.
It provides a centralized system to handle crops, livestock, payments, and sales activities.

---

## 💻 Technologies Used

* HTML
* CSS (Bootstrap)
* JavaScript
* PHP
* MySQL
* XAMPP
* Docker

---

## ⚙️ Features

* Crop management system
* Livestock tracking
* Payment and billing module
* Dashboard with statistics
* Sales and marketing reports

---

## 🚀 Getting Started

### 🔹 Run using XAMPP

1. Start Apache and MySQL in XAMPP
2. Move the project folder to `htdocs`
3. Open in browser:
   http://localhost/DPFMS

---

### 🔹 Run using Docker

Run the following command inside the project directory:

```bash
docker-compose up --build
```

Then open in browser:
http://localhost:8080

---

## 📁 Project Structure

* `index.html` – Main page
* `dashboard.html` – Dashboard
* `payment.html` – Payment module
* `*.php` – Backend logic
* `css/` – Stylesheets
* `js/` – Scripts
